# Cisco Packet Tracer - Static Routing Lab

## Objective
Configure static routing between three Cisco routers.

## Devices Used
- 3 Cisco Routers
- 3 Switches
- 6 PCs

## Skills Learned
- IP Addressing
- Static Routing
- Serial Connections
- Router Configuration
- Network Troubleshooting
- Connectivity Testing

## Commands Used
- ip address
- no shutdown
- ip route
- show ip route
- ping

## Result
Successfully established communication between all PCs using static routing.
